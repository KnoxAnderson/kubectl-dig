# kubectl dig

> ...

## Install

```bash
go get -u github.com/leodido/kubectl-dig/cmd/kubectl-dig
```

## Usage

The format is ...

```
kubectl dig <node>
```

For example.

```
kubectl dig ip-180-12-0-152.ec2.internal
```
